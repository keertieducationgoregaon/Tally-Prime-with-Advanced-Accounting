1. Theory Section

1.1 What Are Cost Centres?

Any unit (department, project, branch, employee) whose income & expenses you want to track separately.

  • Project Alpha, Project Beta
  • Sales Dept, Production Dept
  • Mumbai Branch, Delhi Branch

1.2 Why Use Cost Centres?

  • Know which project is profitable
  • Identify high-cost departments
  • Compare branch performance
